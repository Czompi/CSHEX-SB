# CSHEX-SB
## Endpoints
### POST `/v1/convert`
#### Request
```json
{
    "url": "https://domain.hu/to/be/converted"
}
```
#### Response
A generated PDF, that can be edited by any PDF editor and it consumes less space, than a Chrome/Edge *Print to PDF* file.
